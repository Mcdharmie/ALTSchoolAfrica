Provisioning a Linux Server with a Simple Landing Page

About the Project

This project demonstrates the process of provisioning a Linux server, setting up a web server, and deploying a custom HTML landing page. It showcases the foundational skills necessary for cloud server management and web development while highlighting the potential for scalable solutions.

Project Objectives

  • Provision a Linux server on a virtualization or cloud platform (e.g., AWS).
  • Install and configure a web server (Apache or Nginx).
  • Deploy a custom HTML landing page with detailed project information.
  • Configure the server for public access via HTTP.

Outcomes

The project successfully provides a live demonstration of technical capabilities by deploying a landing page accessible via the server's public IP. This serves as a proof of concept for a scalable web application.

Future Scope

Future enhancements could include adding HTTPS support using SSL certificates, integrating dynamic content with backend services, and expanding functionality to support user interactions.
